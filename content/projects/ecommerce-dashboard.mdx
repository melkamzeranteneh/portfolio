## Project Overview

The E-Commerce Dashboard is designed to verify real-time data handling and visualization skills. It connects to a Supabase backend to display live order updates.

### Key Features

- **Real-time Analytics**: Visualized using Recharts.
- **Product Management**: CRUD operations for inventory.
- **Order Tracking**: Live status updates for customer orders.

### Technical Challenges

Handling state synchronization between the dashboard and the database was the primary challenge, solved using Supabase Realtime subscriptions.
